Overriding

This library only exposes the most common customizations of HighCharts chart types. This is for ease of use and brevity. Nice charts with minimal code.

However, you can always pass in an 'options' attribute for advanced HighCharts features.

The value of this attribute will be merged into the chart's options created by this library just before creating the chart. In combination with the 'debug' attribute, you can see how you are changing the options that create the HighChart chart.
